TabFolder/TabItem

A TabFolder is very similar to a notebook type GUI and allows the user to select a page from a set of notebook pages... or switch to a page within the notebook. The children of the TabFolder can only be TabItems and is the notebook page object. Controls can be created and set into each TabItem using TabItem.setControl.

Provide the following capability:

  • Controls TabFolder/TabItem relationship in the model
  • Decoder to handle TabFolder/TabItem model.
  • Graphical edit part for TabFolder.
  • Context menu for switching between the pages in the notebook.
  • Internal layout policy to handle dropping controls onto TabFolder
  • Tree edit part that interacts in the Beans Viewer
  • Property sheet... identify style properties to be shown
  • Direct edit support to set the TabItem text
  • Target VM manager to handle add, delete, and reordering of the children.
