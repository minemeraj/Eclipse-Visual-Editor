CoolBar/CoolItem

Instances of CoolBar provide an area for dynamically positioning the items it contains. The item children that may be added to instances of CoolBar must be a CoolItem. Controls can be set into the CoolItems using CoolItem.setControl.

Provide the following capability:

  • Controls CoolBar/CoolItem relationship in the model
  • Decoder to handle CoolBar/CoolItem model.
  • Graphical edit part for CoolBar.
  • Internal layout policy to handle dropping controls onto CoolBar
  • Tree edit part that interacts in the Beans Viewer
  • Property sheet... identify style properties to be shown
  • Target VM manager to handle add, delete, and reordering of the children.
