Shell

Top level SWT widget in which instances of this class represent the "windows" which the desktop is managing.

Provide the following capability:

  • Target VM (off screen) capability
  • Decoder
  • Graphical edit part (image from target VM) which handles a client area
  • Tree edit part that interacts in the Beans Viewer
  • Menubar support...including modeling and decoder
  • Toolbar support... modeling, decoder
  • Property sheet... identify styles properties to be shown
