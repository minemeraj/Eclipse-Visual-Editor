Tree/Tree Item

Instances of this class provide a selectable GUI object that displays a hierarchy of items and issues notification when an item in the hierarchy is selected. The children must be of type TreeItem.

Provide the following capability:

  • Controls Tree/TreeItem relationship in the model.
  • Decoder to handle Tree/TreeItem model.
  • Graphical edit part for Tree.
  • Tree edit part that interacts in the Beans Viewer.
  • Property sheet... identify style properties to be shown.
  • Direct edit support to set the TreeItem text.
