Visual Editor 1.0.0 Line Item Assignments

The following list represents the current developers that are working on a particular line item.  These small teams are responsible to drive and implement the line item design/code.  This design will be sent to the VE mailing list for open discussions.

 

Submit your request to the VE development mailing list (ve-dev@eclipse.org) for the specific line item(s) you would like to develop and contribute.
