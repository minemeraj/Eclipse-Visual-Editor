Visual Editor Translations
Translation Packs for VE

IBM has contributed translations of Eclipse Tool Project VE.

1.0

The language pack is distributed as a feature which you can install by downloading the zip file, unzipping it into your Eclipse install directory, and restarting Eclipse. The language packs contain translations for German, Spanish, French, Italian, Japanese, Korean, Portuguese (Brazil), Traditional Chinese and Simplified Chinese.

These translations are based on the VE 1.0 build, but should work with all subsequent 1.0 maintenance releases. If new Strings are added to VE after 1.0, they will not show up as translated in the 1.0.x stream when you install this language pack.

1.1.0.1

The language pack is distributed as a two zip files per group which you can install by downloading the zip files, unzipping them into your Eclipse install directory, and restarting Eclipse. 

These translations are based on the VE 1.1.0.1 build, but should work with all subsequent 1.1 maintenance releases. If new Strings are added to VE after 1.1.0.1, they will not show up as translated in the 1.1.x stream when you install this language pack.
