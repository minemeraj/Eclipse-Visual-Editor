Eclipse Visual Editor Project 1.1.0.1 - New and Noteworthy
Support for SWT Menus
 The Visual Editor now provides support for SWT Menus.

 

Menubar can now be dropped onto a Shell, and MenuItems onto the MenuBar

Visual Tab selection in CTabFolders

 The Visual Editor now provides the direct selection of the tabs in a CTabFolder, without having to use the context menu, or the Java Beans outline view.
Ability to reparent SWT containers
 The Visual Editor now allows the reparenting of SWT containers.
Support for 64-bit SWT GTK
 The Visual Editor now supports creating SWT widgets on 64-bit systems with GTK.
