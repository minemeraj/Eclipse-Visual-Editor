The following articles have been written by the Visual Editor Team or other members of the Eclipse community.  Contributions are welcome and encouraged. Ask on the mailing list for information.
Extending the Visual Editor Tutorial: Enabling support for a custom widget  Gili Mendel, Joe Winchester and Dave Orme cover an example where a user has a custom widget that they wish to extend the Visual Editor to have specific builder behavior for.

The Ultra Light Component Model provide server side API for swing like components.  In this article, you'll get an overview of how Canoo extended the VE to build a ULC GUI builder.
