Java Core
Core Visual Java Editor Component
 
Description
The core Java component is the component that supplies the core/base Visual Editor for Java. This is the basic and doesn't include specific extensions, such as for the JFC/Swing. Basic functions are editing a Java class, handling property editors, the basic EMF model of a Java object, the BeansList outline viewer, the base graphical viewer, and the interfaces with the JDT model and editor.
