Deleting a visual bean

To delete a bean using the Visual Editor for Java, do one of the following:

  • Select the bean in the Design view. A heavy black border indicates that the bean is selected. Press the Delete key.
  • Select the bean's entry in the Java Beans view ( Window > Show View > Java Beans) or in the Design view. Press the Delete button on the toolbar, or press the Delete key.
  • Select Delete from the pop-up menu in the Java Beans view or the Design view

If more than one Java bean is selected, then the Delete key or action will remove all of the selected beans. After deletion, the Java source code that is displayed in the source view of the Visual Editor is updated to reflect the removal of the bean, and the Java Beans view and the Design view are refreshed
