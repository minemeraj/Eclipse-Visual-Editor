Specifying the location of BeanInfo classes

The classes that you develop in the Visual Editor for Java must be stored in Java projects. Each Java project has a build path that includes references to other projects inside your IDE workspace or external JAR files that exist in the file system. The packages included in the build path are used to compile the classes in the project, and they are also used by the Visual Editor for Java to create the class path of the virtual machine where runs the Java beans at design time.
