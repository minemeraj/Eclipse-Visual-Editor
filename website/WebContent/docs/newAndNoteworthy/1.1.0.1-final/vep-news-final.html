Eclipse Visual Editor Project 1.1.0.1 - New and Noteworthy
Support for SWT Menus
 The Visual Editor now provides support for SWT Menus. (106694, 108031)

 

Menubar can now be dropped onto a Shell, and MenuItems onto the MenuBar

Visual Tab selection in CTabFolder and JTabbedPane

 The Visual Editor now provides the direct selection of the tabs in SWT's CTabFolder and Swing's JTabbedPane, without having to use the context menu, or the Java Beans outline view. (100569)

CTabFolder

 JTabbedPane

Ability to reparent SWT containers
 The Visual Editor now allows the moving of an SWT container from one container to another. However  non-container widgets cannot be moved from one container to another.
Support for 64-bit SWT GTK
 The Visual Editor now provides preliminary support for creating SWT widgets on 64-bit systems with GTK. (97769)
Bug fixes and enhancements
 The Visual Editor has fixed 79 defects along with providing 2 enhancements for the 1.1.0.1 release.
