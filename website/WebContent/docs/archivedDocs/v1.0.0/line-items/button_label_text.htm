Button, Label, Text

Button - Instances of this class represent a selectable GUI object that issues notification when pressed or released. The styles that may be specified are ARROW, CHECK, PUSH, RADIO, TOGGLE, FLAT, UP, DOWN, LEFT, RIGHT, or CENTER.

Label - Instances of this class represent a non-selectable GUI object that displays a string or image. When the style SEPARATOR is specified, it will display a single vertical or horizontal line. The styles that may be specified are SEPARATOR, HORIZONTAL, VERTICAL, SHADOW_IN, SHADOW_OUT, SHADOW_NONE, CENTER, LEFT, RIGHT, or WRAP.

Text - Instances of this class represent a selectable GUI object that allows the user to enter and modify text. The styles that may be specified are CENTER, LEFT, MULTI, SINGLE, RIGHT, READ_ONLY, or WRAP.

Provide the following capability:

  • Basic graphical and tree edit part for Control.
  • Property sheet... identify style properties to be shown
  • Direct edit support to set the text
