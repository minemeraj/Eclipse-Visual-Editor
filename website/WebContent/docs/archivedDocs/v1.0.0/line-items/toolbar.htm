ToolBar/ToolItem

Instances of this class provide a GUI interface of selectable tool bar items. Only a ToolItem can be a child.

Provide the following capability:

  • Controls ToolBar/ToolItem relationship in the model
  • Decoder to handle ToolBar/ToolItem model.
  • Graphical edit part for ToolBar.
  • Tree edit part that interacts in the Beans Viewer
  • Property sheet... identify style properties to be shown. Styles specified for ToolItem can only be CHECK, PUSH, RADIO, SEPARATOR, or DROP_DOWN.
  • Direct edit support to set the ToolItem text
  • Target VM manager to handle add, delete, and reordering of the children.
