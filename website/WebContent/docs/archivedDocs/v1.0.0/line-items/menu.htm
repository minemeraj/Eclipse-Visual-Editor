Menu/MenuItem

Instances of the Menu class are GUI objects that contain menu items. Construction of this class can be with Control, other menus, or other menu items to create cascading menus. It's children are of type MenuItem. Valid styles for Menu are BAR (to create a menu bar), DROP_DOWN, or POP_UP. Valid styles for MenuItem are CHECK, PUSH, RADIO, or SEPARATOR. MenuItems and Menus are selectable GUI objects that issue notifications when pressed and released.

Provide the following capability:

  • Controls Menu/MenuItem relationship in the model.
  • Decoder to handle Menu/MenuItem model.
  • Encapsulate menu bar, popup menu, and cascade menu capability as a separate controls.
  • Graphical edit part for Menu.
  • Internal layout policy to handle dropping controls onto TabFolder
  • Tree edit parts for Menu and MenuItem that interacts in the Beans Viewer
  • Property sheet... identify style properties to be shown
  • Direct edit support to set the Menu and MenuItem text.
  • Target VM support with menu manager to re-order menu items as needed.
