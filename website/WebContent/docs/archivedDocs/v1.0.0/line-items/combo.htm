Combo

Instance of this class allow a user to choose from a list of items, or optionally enter a new value by typing it into an editable field. A Combo takes less space that a List and is very useful where GUI space is limited. The styles that may be specified are DROP_DOWN, READ_ONLY, or SIMPLE.

Provide the following capability:

  • Basic graphical and tree edit part for Control.
  • Property sheet... identify style properties to be shown
